# Simple Music Button Site

Este é um projeto simples de site que apresenta um botão centralizado na tela. Quando o botão é clicado, ele toca uma música gratuita (do SoundHelix) e muda seu estado para "Pausar". Clicar novamente pausa a música.

## Características

- **Design moderno e elegante**: Utiliza um degradê de fundo suave e um botão com aparência fosca e translúcida.
- **Paleta de cores específica**:
  - Degradê de fundo: de `#215B63` (topo) para `#124170` (fundo)
  - Textos e bordas: `#AAFFC7`
- **Botão interativo**: 
  - Formato retangular com bordas arredondadas
  - Efeitos de hover e transições suaves
  - Muda de texto e estilo ao tocar/pausar a música
- **Funcionalidade de áudio**: 
  - Toca uma música gratuita do SoundHelix ao clicar no botão
  - Permite pausar e retomar a reprodução
- **Responsivo**: Funciona bem em dispositivos móveis e desktop.
- **Organização do código**: 
  - Arquivos separados para HTML, CSS e JavaScript
  - Comentários explicativos em cada arquivo

## Estrutura do Projeto

- `index.html`: Estrutura básica da página
- `style.css`: Todos os estilos, incluindo o degradê de fundo e o design do botão
- `script.js`: Lógica para controlar a reprodução de áudio e o estado do botão
- `.gitignore`: Ignora o arquivo `.openclaude-profile.json`

## Como Usar

1. Clone este repositório ou baixe os arquivos
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Clique no botão para tocar a música
4. Clique novamente para pausar

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis, degradês e efeitos de blur)
- JavaScript ES6

## Créditos

- Música utilizada: "SoundHelix-Song-1.mp3" do [SoundHelix](https://www.soundhelix.com/) (gratuita para uso)
- Ícones: Utilizados caracteres Unicode para play (▶) e pause (⏸)
